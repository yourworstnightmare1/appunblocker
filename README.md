## Welcome to appUnblocker

appUnblocker allows you to open apps on Windows that have a UAC prompt (requires username/password to open) without the login credentials.
<br>

Can be used on most installers and apps with the `.exe` or `.msi` extension.
Example: `steam.exe` / `ReBlockInstaller.msi`
